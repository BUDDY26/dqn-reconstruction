## Notes

These values are copied directly from the original paper and must not be modified.
They serve as the reference baseline for comparing results generated by the
reconstructed implementation.

Verification results from the reconstructed model will be saved to:

```
results/verification_results.json
```

The purpose of this file is to clearly distinguish three categories of results:

1. **Paper-reported results** — this file; original paper baseline
2. **Reconstructed results** — `results/verification_results.json`; generated by running `scripts/run_verification.py`
3. **Improved model results** — future work on the `improvement-experiments` branch

---

## Metric Definitions

As used in the paper and implemented in `src/evaluate.py`:

| Metric | Definition |
|--------|-----------|
| ROI | Total return over the test period |
| Sharpe | Annualized Sharpe ratio (risk-free rate = 0, 252 trading days) |
| Max Drawdown | Largest peak-to-trough portfolio decline |
| Calmar | Annualized return divided by maximum drawdown |
